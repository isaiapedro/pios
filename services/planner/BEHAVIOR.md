# Engineering Principles
- **Autonomy Preservation:** Code constructs must treat schedules as flexible, protective opportunities. The system must never enforce mandatory checkboxes.
- **No LLM Direct Mutability:** Language models must never obtain unrestricted write access to calendar repositories or state databases.
- **Loud Verification Halts:** If a proposed block intersects with an existing obligation or sleep constraint, the pipeline must immediately abort and generate explicit error traces.
- **Deterministic Analytics:** Metrics can only be computed using deterministic database aggregations. AI handles contextual narrative interpretation, never computational math.
