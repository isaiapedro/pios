# Personal Planner

## Purpose

Represent the complete personal state of the user.

This domain stores observations,
evidence, reflections,
plans and derived knowledge
about the user's life.

It is the canonical source of
subjective information within PIOS.

## Responsibilities

- Maintain canonical personal state.
- Integrate evidence into coherent context.
- Produce derived personal knowledge.
- Preserve historical integrity.
- Provide context to services.

## Inputs

- Voice transcripts
- Calendar
- User input
- Reviews

## Outputs

- Plans
- Insights
- Reviews
- Recommendations

## Non-responsibilities

- Scientific truth
- Vendor documentation
- Service implementation
