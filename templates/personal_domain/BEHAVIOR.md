# Personal Domain Template — Behavior Specification

## 1. Intentional Parameters
- Designed exclusively for self-directed life development tracking (e.g., finance, habits, long-term focus mapping).
- Protects personal data streams by assigning explicit privacy and semantic flags before exposure to intelligent parsing loops.

## 2. Execution Flexibility
- Never enforces a task-level execution loop. Instead, it curates high-level soft rules to guide Layer 3 scheduling.
- Output data must prioritize semantic integration over tactical checkpoints.
