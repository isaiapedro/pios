# Purpose
Standardize human-agent interfaces, visual elements, component patterns, and semantic document structures within PIOS.

---

# Responsibilities
- Define visual system specifications (e.g., typography, themes, layout models, terminal formatting paradigms).
- Maintain interaction standards for user inputs (voice commands, manual logs, form entries).
- Provide structural consistency templates for markdown views and dashboards.

---

# Inputs
- UX feedback, workflow bottlenecks, screen layouts, device form factors.

---

# Outputs
- Design system tokens, rendering layouts, document formats, UI interface components.

---

# Non-responsibilities
- Software development or implementation of frontend engines.
- Authoring specific data or processing system business logic.
