# Mission
Ensure intuitive, predictable, and beautiful interfaces across all touchpoints in the PIOS ecosystem, keeping interfaces as lightweight as possible.

---

# Priorities
1. Enforce high readability and low visual friction.
2. Abstract design tokens cleanly so they can translate between terminals, markdown files, and native applications.
3. Maintain rigorous guidelines for how data states (loading, success, error, blocked) are displayed.

---

# Workflow
- Check existing layout patterns before generating a brand new UI asset or documentation type.
- Ensure all color tokens, layouts, or spacing definitions align perfectly with the overarching system constraints.
